# Debug Workflow 执行清单

复制以下清单并逐项完成：

Task Progress:

- [ ] 1. 复述现象、触发条件、预期行为
- [ ] 2. 选择最小复现路径并记录命令/步骤
- [ ] 3. 标记不确定信息与当前假设
- [ ] 4. 追踪入口与关键调用链，确认根因
- [ ] 5. 设计最小改动方案并评估影响面
- [ ] 6. 实施修复，不做无关重构
- [ ] 7. 执行贴近问题的验证
- [ ] 8. 补充类型检查/构建（如可执行）
- [ ] 9. 输出：根因、修改点、验证结果、剩余风险

## 最低交付标准

- 至少 1 个可复现证据
- 至少 1 个修后验证证据
- 明确是否存在未验证项
